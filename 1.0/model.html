class Starfish.model.Model(DataSpectrum, Instrument, Emulator, stellar_tuple, cheb_tuple, cov_tuple, region_tuple, outdir='', max_v=20, ismaster=False, debug=False)[source]¶

Container class to create and bring together all of the relevant data and models to aid in evaulation.

Parameters:
  • DataSpectrum (spectrum.DataSpectrum object) – the data to fit
  • Instrument (grid_tools.Instrument object) – the instrument with which the data was acquired
  • HDF5Interface (grid_tools.HDF5Interface object) – the interface to the synthetic stellar library
  • stellar_tuple (tuple) – describes the order of parameters. If alpha is missing, :obj:grid_tools.Interpolator is trilinear.
